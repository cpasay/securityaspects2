Security Aspects

(GCSE Computer Science)

1. Malware (Malicious Software)

  • A software that is designed to damage/control computer systems
  • E.g. viruses, worms. Trojan viruses, spyware, adware and ransomware

2. Virus

  • A computer virus needs user input to replicate itself. It replicates itself by modifying other computer programs and inserting its own code. While a computer worm is able to self-replicate.
  • Affected areas are said to be “infected” with a computer virus

3. Spyware

  • A software that can gather data about an individual.
  • 4 Categories: Adware, System Monitors, Tracking, Trojans
  • E.g. keyloggers, rootKits, web beacons

4. Hacking

  • Trying to gain unauthorized access to a computer system
  • White hat: Gaining access and advice
  • Black hat: Gaining access to do damage

5. DoS or DDoS

More info
  • Denial of Service or Distributed Denial of Service attack floods a server with requests and disrupting its service through the volume of traffic.

6. Firewalls

  • Can be software, hardware or both. (Packet filters, stateful inspection and proxy server firewalls.

7. Anti Virus Software

  • Created specifically to help detect, prevent and remove malware
  • Runs automatically in the background to provide real-time protection against virus attacks once installed

8. Proxy Server Firewall

Know more
  • A good proxy server can keep the users on the network safe by filtering out unsafe content
  • Maintains your privacy as the proxy server will have a different IP address from you
